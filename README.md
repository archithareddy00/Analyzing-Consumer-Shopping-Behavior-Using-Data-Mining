# Analyzing-Consumer-Shopping-Behavior-Using-Data-Mining
# Online Retail Sales Analysis

This project analyzes customer shopping behavior using Python, SQL, and Looker Studio.


## Dataset

This project uses the Online Retail dataset from Kaggle:
[https://www.kaggle.com/datasets/tunguz/online-retail](https://www.kaggle.com/datasets/tunguz/online-retail)

## Files Included

* Jupyter Notebook for data cleaning and analysis
* Dashboard screenshot
